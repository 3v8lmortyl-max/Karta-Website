# Karta — Wear Art, Wear Karta

> Premium handmade artistic fashion brand website

## Brand
**Karta** creates handmade artistic designs on quality fabric. Every piece is wearable art.  
*Motto: Wear Art, Wear Karta*

## Tech Stack
- **Frontend:** Next.js 14 (App Router) + Tailwind CSS
- **Database & Auth:** Supabase
- **Payments:** Razorpay
- **Deployment:** Vercel
- **Media:** Cloudinary (product images)

## Features
- 🛍️ Product catalog with collections
- 🛒 Cart & checkout
- 👤 User accounts (sign up, login, order history)
- 🎨 Admin panel (catalog management)
- 💳 Online payments via Razorpay
- 📱 Mobile-first responsive design

## Pages
- `/` — Home (hero, featured collections, brand story)
- `/shop` — Full catalog with filters
- `/collections/[slug]` — Collection page
- `/products/[slug]` — Product detail
- `/cart` — Cart
- `/checkout` — Checkout
- `/account` — User account & orders
- `/about` — Brand story
- `/contact` — Contact
- `/admin` — Admin panel (protected)
